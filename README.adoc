== ğŸ§ª Model Details

Model: ResNet18 (Pretrained on ImageNet)

Transfer Learning approach:
- Freeze early layers
- Replace final FC layer
- Fine-tune classifier

Loss Function:
- CrossEntropyLoss

Optimizer:
- Adam

Epochs:
- 5

Device:
- GPU (NVIDIA T4 on Colab) for training
- CPU for inference
